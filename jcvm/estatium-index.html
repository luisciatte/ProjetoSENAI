Sobre nós

Somos a JCVM Tech, uma empresa formada por 5 membros: Maria Eduarda (Presidente - Scrum Master), Víctor Gabriel (Design Gráfico - Front End Team Scrum), João Vitor (Banco de Dados - Team Scrum), Camila Elisa (Responsável pelo Marketing) e Wallace (Back End - Team Scrum).

A JCVM Tech é uma empresa fundada com o objetivo de transformar o que está no papel em realidade. Trabalhamos com o desenvolvimento de MRP para suporte de PCP e chão de fábrica, visando sempre a praticidade e clareza para cada usuário.

Proporcionamos um crescimento sustentável e competitivo a longo prazo, na qual é possível planejar: quando, quanto, onde produzir e qual o destino do produto final. Quem vai com tudo, vai além.

Linguagens de programação

Na Estatium, exploramos as linguagens de programação essenciais que impulsionam a criação de soluções digitais inovadoras.

Desde a estruturação básica com HTML até a interatividade dinâmica com JavaScript, cada linguagem desempenha um papel crucial na construção de experiências web robustas e funcionais.

HTML

HTML permite definir a estrutura básica de uma página, incluindo elementos como texto, imagens e links.

CSS

CSS é usado para controlar o layout, as cores e os estilos de todos os elementos HTML em um site.

JavaScript

JavaScript permite criar funcionalidades avançadas, como animações, validações de formulários e atualizações de conteúdo em tempo real.

Java

Java é amplamente utilizado em aplicativos corporativos, desde sistemas financeiros até aplicativos móveis.

SQL

SQL é essencial para consultas complexas, inserções de dados e operações de atualização em bancos de dados..

JSP

JSP é uma forma de implementar o conteúdo da página dinâmica do lado do servidor.

Nosso objetivo

A JCVM Tech visa um bom desempenho do maquinário. Somos especialistas em desenvolver MRP para suporte de PCP e chão de fábrica, de forma clara e prática para os usuários. Temos o objetivo de transformar o que está no papel em realidade. Buscamos a otimização de processos e aumento da produtividade, redução de custos e otimização de recursos, inovação e vantagem competitiva e segurança da informação e proteção de dados. Nosso time

Nossos valores

  1. Inovação: Comprometimento com a criação de soluções tecnológicas avançadas que atendam às necessidades em constante evolução da indústria têxtil;

  2. Qualidade: Foco incessante na entrega de produtos e serviços que superem as expectativas dos clientes;

  3. Transparência: Manter uma comunicação clara e honesta com nossos clientes, parceiros e colaboradores;

  4. Sustentabilidade: Promover práticas que minimizem o impacto ambiental e contribuam para o desenvolvimento sustentável;

  5. Excelência no Atendimento: Prover um suporte excepcional, garantindo que nossos clientes obtenham o máximo benefício de nossas soluções.
